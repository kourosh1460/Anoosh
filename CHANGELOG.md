# Changelog

## Desktop 2.2.0 · Android 1.1 — 2026-07-12

- Three new themes on both apps: **Toranj** (natural woods palette, in a
  Persian teal-blue and a burnt-orange variation — paper-grain background,
  serif titles, framed cards) and **Blossoms** (pink & violet night garden
  with petal-shaped checkboxes and controls).
- New theme gallery picker with live color previews.
- Android: **multi-select for notes** — long-press a note inside a folder,
  then bulk move, favorite or delete.

## Desktop 2.1.x · Android 1.0 — 2026-07-11

- **Two-way sync** between desktop and phone: over Wi-Fi (address + 6-digit
  code) or by file for cable transfers. Real merge with tombstones — never a
  blind replace; deletions propagate, newer edits win, settings stay
  per-device.
- First Android release: full app (tasks, notes in folders with images,
  ideas, tri-calendar, reminders as native notifications, timer), home-screen
  **Today widget**, three themes.
- Fixed: Settings screen failing to open after a bad merge of the sync panel
  code; sync port lingering after stop.

## Desktop 2.0.0 — 2026-07-11

- **Jalali (solar) calendar** joined Gregorian and Hijri — full three-way
  conversion, any system can drive the month grid.
- **Note folders** with colors and pinning; the Notes tab now opens on a
  folder grid.
- **Images in notes** — paste, drop or insert; stored inline and downscaled.
- **Onyx** theme (solid near-black, zero blur).
- Renamed from Aurora to Anoosh; automatic data migration.
- Responsive fixes at the minimum window size.

## Desktop 1.0.0 — 2026-07-11

- First release: tasks, notes, ideas board, Gregorian + Hijri calendar,
  reminders with native notifications, pomodoro timer, cross-linking between
  every module, dark/light themes, screen-pinned floating notes, Windows
  installer with auto-start.
